Votka [![Status Zero][status-zero]][andivionian-status-classifier]
=====

Votka is a .NET Standard library for parsing of [VTK file format][vtk-format].

Build
-----

[.NET Core 2.0 SDK][net-core-sdk] is required to build the project.

```console
$ dotnet build
```
